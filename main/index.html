ESP32-IDF Профи Вызов

Битва инженеров на чистом Си в ESP-IDF v5.5.4

О конкурсе

Вас ждет серия из 5 сложных этапов. Мы будем шаг за шагом собирать и улучшать проект умного устройства. Весь код пишется строго в рамках операционной системы FreeRTOS.

  • Этап 1: Управление RGB-светодиодами на разной частоте через потоки.
  • Этап 2: Сбор данных с датчиков DHT22 и BH1750 через очереди.
  • Этап 3: Создание надежного конечного автомата (FSM) для управления системой.
  • Этап 4: Подключение к Wi-Fi сети и обработка обрывов связи через LwIP.
  • Этап 5: Отправка данных по MQTT в формате JSON с обработкой ошибок.

Условия участия

Правила строгие, но честные. Все участники получают одинаковые стартовые условия.

  • Только ESP-IDF: Пишем на чистом Си. Никакой среды Arduino!
  • Версия ПО: Используем строго ESP-IDF v5.5.4 и редактор VS Code.
  • Сборка: Проект должен собираться без ошибок через CMake.
  • Железо: Все работают на одинаковых платах ESP32-C3 и датчиках.
  • Сдача кода: Готовая работа отправляется в виде Pull Request на GitHub.

Балльная система
